QUE ES HTML?

HTML es la sigla en ingles HyperText Markup Language, refiere al lenguaje del marcado en la elaboración de páginas web o también, se define como el lenguaje para crear documentos Web, es la base de la construcción de cualquier página Web, pues los diferentes navegadores (chrome, firefox, edge, safari) solamente comprenden este lenguaje. Nos referimos a lenguaje como el conjunto de elementos que, ubicados de forma logica, dan significado a otro elemento, por tanto, HTML es un lenguaje de marcas, que usan una serie de etiquetas estandarizadas. HTML 5 es la versión más reciente, liberada a finales de octubre del 2014, incorpora nuevos elementos, para introducir multimedia de forma directa en la web sin necesidad de plugins.
